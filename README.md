Este repositorio contiene el código fuente, scripts de configuración, esquemas de base de datos, ejemplos de pruebas y documentación técnica del proyecto titulado:
“Conversión de instrucciones en lenguaje natural a consultas SQL en bases de datos transaccionales para personal no técnico.”

El objetivo del proyecto es implementar un agente conversacional inteligente que interprete instrucciones en lenguaje natural y las transforme automáticamente en consultas SQL ejecutables sobre una base de datos estructurada.

La solución fue desarrollada utilizando herramientas de inteligencia artificial generativa y servicios en la nube, incluyendo:

Un modelo de lenguaje aprovisionado con Azure AI Foundry

Una base de datos simulada en Azure SQL Database

Una interfaz web construida con Flask y desplegada en Azure App Services

Este trabajo está orientado a resolver una necesidad común en instituciones financieras y otras organizaciones que manejan datos operativos, permitiendo que personal no técnico acceda a información crítica sin necesidad de conocimientos en SQL.

📂 #Estructura del repositorio
/app/ – Código fuente de la aplicación Flask

/sql/ – Scripts de creación y carga de datos en la base de datos

/prompts/ – Instrucciones de entrada utilizadas para entrenamiento o pruebas

/docs/ – Diagramas, presentación de arquitectura y guía de uso

/tests/ – Casos de prueba con entradas y salidas esperadas
